# Princess Learning Adventure ✨

A Disney-princess-themed learning game (Ariel, Belle, Cinderella, Elsa, Jasmine, and friends).
A few things to do:

- 🔤 **Letters** — Letter Tracing, Letter Explorer, and Princess Letters activities, working up
  through Letter Champion and Letter Master
- ✍️ **Spelling** — Spelling Practice building up to Spelling Star and Spelling Master
- 🧠 **Memory Game** — matching rounds that build up to Memory Palace and Memory Master
- 👗 **Dress Up** — pick a princess friend and dress her up (Green Dress, Purple Dress, Mermaid
  Dress, and more)

## How to open it

Open **`index.html`** in any web browser (Chrome, Edge, Safari). **Needs an internet connection**
the first time it loads — it pulls React, Babel, and Tailwind from a CDN — but works fine on a
laptop or tablet once loaded.

## For grown-ups

Built as a single-file React app (JSX compiled in-browser via Babel standalone), styled with
Tailwind CSS loaded from a CDN.
